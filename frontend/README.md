# Frontend

React + Vite dashboard for Neural Snake.

## Purpose
- Show live training metrics
- Start/stop/reset training
- Display episode charts and checkpoints
- Control the FastAPI backend

## Setup
1. Install Node.js
2. Run:
   ```bash
   npm install
   npm run dev
   ```

## API connection
Point the frontend to the FastAPI backend at `http://localhost:8000`.
